# Resource Production

City production is defined directly by the resource-building property tables. Food and Wood start at **50,000**, Iron and Mithril start at **0**, and Gold starts at **100**.

## Starting resources

| Resource | Starting amount |
| --- | --- |
| Food | 50,000 |
| Wood | 50,000 |
| Iron | 0 |
| Mithril | 0 |
| Gold | 100 |
