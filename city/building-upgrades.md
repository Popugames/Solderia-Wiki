# Building Upgrades

## Overview
Most buildings can be upgraded to improve their function and unlock stronger progression.

## Requirements
An upgrade may require:
- a minimum Castle level
- another building at a required level
- enough resources
- a free construction queue if your current rules use one

## Time
Building upgrades take time and continue even while you are offline.

## What happens when an upgrade starts
When you start an upgrade:
- the resource cost is consumed
- the timer begins
- the building enters an upgrade state
- the result becomes available only when the timer finishes

## What is allowed
You may:
- start an upgrade if all conditions are met
- complete it naturally over time
- use a speedup if the game allows it

## What is not allowed
You may not:
- start an upgrade without enough resources
- ignore building prerequisites
- claim the upgrade result before the timer finishes
