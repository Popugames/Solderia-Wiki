# Wishing Well Rules and Restrictions

## What is allowed
Players may:
- unlock the Wishing Well
- upgrade it if it is a normal building
- spend a valid charge or attempt to claim a reward
- use a refresh if they still have a valid refresh available
- benefit from the Wishing Well while respecting all limits

## What is not allowed
Players may not:
- claim rewards without a charge or attempt
- claim the same reward twice
- refresh without meeting the refresh rule
- use the Wishing Well as an unlimited economy source
- reset its rules by changing device time

## Special cases
If daily reset happens before a claim, the system uses the new valid daily state.
If a claim request is sent twice, only one valid result should be applied.
