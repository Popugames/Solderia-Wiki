# Gathering on the Map

## Overview
Gathering on the world map is different from passive city production. It requires active troop use.

## Basic process
- send troops to a node
- wait for travel
- wait for gathering
- wait for return

## Important factors
Gathering amount depends on:
- node amount
- troop load
- gathering speed
- time spent gathering
- possible buffs

## Restrictions
Gathering cannot continue forever if:
- the node is exhausted
- troop capacity is reached
- the troops are recalled
