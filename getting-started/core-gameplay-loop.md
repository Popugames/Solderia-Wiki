# Core Gameplay Loop

The normal gameplay loop in Solderia is:

1. produce and collect resources
2. upgrade buildings
3. research improvements
4. train troops
5. send marches to the world map
6. gather resources or attack monsters
7. grow power
8. join alliance activities
9. fight for control of major objectives

## Why this matters
Every major system in the game supports this loop. If you understand this sequence, you understand the core of the game.

## Progress pacing
Early game focuses on:
- Castle growth
- resource production
- unlocking key buildings

Mid game focuses on:
- stronger troops
- more research
- efficient map usage
- alliance coordination

Late game focuses on:
- contested objectives
- alliance warfare
- Palace control
