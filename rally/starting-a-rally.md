# Starting a Rally

## Who can start a rally
Only players with the proper alliance permissions can start a rally.

## How it works
When a rally is created:
- a target is selected
- a formation timer begins
- alliance members may join during the formation phase

## Rally timer
The formation timer gives members time to contribute troops before the rally launches.

## Requirements
To start a rally, the player must:
- be in an alliance
- have the required permission
- have troops available
- select a valid target
- respect any rally limits that apply

## What is not allowed
A player cannot start a rally:
- without permission
- without available troops
- against an invalid target
- if another blocking rally limit is already reached
