# General Rules

This page summarizes the most important player-facing rules.

## Core rule
Every action in Solderia must meet its listed requirements before it can begin.

## Timed action rule
Timed actions continue while you are offline and resolve according to server time.

## Troop rule
The same troops cannot be used in more than one action at the same time.

## Alliance rule
Alliance actions must not leave troops or ownership in an invalid state.

## Palace rule
The Palace is an alliance objective and is not treated like a normal personal target.

## Wishing Well rule
The Wishing Well gives controlled bonus value and is not a replacement for the main economy.
